# WhatsApp UI Clone

> ⚠️ This project is built **purely for educational purposes** as part of my self-learning journey in React/frontend development. It is a UI recreation exercise, not affiliated with, endorsed by, or claiming ownership over WhatsApp or Meta. All trademarks belong to their respective owners.

## Tujuan
Project ini dibuat untuk latihan React fundamentals: component architecture, state management, conditional rendering, routing, dan styling. WhatsApp dipilih sebagai referensi UI karena familiar dan punya kompleksitas interaksi yang pas untuk belajar (tab navigation, chat list, dll).

## Tech Stack
- React + Vite
- React Router
- Express

## Yang Dipelajari
- React Router (`BrowserRouter` vs `createBrowserRouter`)
- Component extraction & props passing
- `useState` untuk tab/nav/menu state
- Conditional rendering (`&&`, ternary)
- CSS layout (`position`, `backdrop-filter`, `calc()`, `100dvh`)
